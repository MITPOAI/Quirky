# Responsible Use Guidelines

Quirky is designed as a local writing-quality alignment layer that aims to improve text readability, remove repetitive AI-generated boilerplate, and restore natural human variation to synthetic media. 

To ensure fair and responsible usage:

## 1. Writing Quality Tooling
Quirky should be used to improve the style, brevity, and formatting of text. It is a tool for developers, authors, and editors to clean up redundant patterns, LLM clichés, and confidence-diluting qualifiers.

## 2. Evasion and Provenance-Disguise
Quirky must **not** be used to bypass academic integrity checks, evade provenance tracking, or hide the synthetic origin of content for deceptive purposes. 

## 3. Probability Scores are Heuristic
The slop scores calculated by Quirky are statistical probabilities derived from heuristic features (e.g. burstiness, type-token ratio, cliché density). They are stylistic indicators, not binary proof of machine authorship. High scores should prompt styling reviews, not immediate accusations of plagiarism.

## 4. Attribution
The `"Powered by Quirky (MITPO)"` attribution notice must be preserved across all output pipelines and media comparisons to maintain provenance and credit the alignment engine.
